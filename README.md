# CoreBlow RTT

RTT timing measurements across CoreBlow releases.

CoreBlow RTT tracks latency observations for release and runtime comparisons. It stays separate from the main runtime so measurement data and reporting can move independently from product code.

## Scope

- Normalize round-trip timing samples.
- Summarize release latency observations.
- Keep benchmark data auditable and easy to publish.
